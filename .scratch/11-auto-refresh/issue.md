## What to build

대시보드가 최신 정보를 자동으로 표시하도록 구현합니다. 1분 간격으로 자동 새로고침되어 메트릭, 서버 상태, 취약점 정보가 실시간으로 업데이트됩니다. 사용자가 원하면 수동 새로고침 버튼으로 즉시 업데이트할 수 있습니다. 점검 실행 중에는 진행 상황을 실시간으로 표시하여 사용자가 점검 상태를 모니터링할 수 있습니다.

**핵심 요소:**
- 1분 간격 자동 새로고침 (useEffect + setInterval)
- 수동 새로고침 버튼
- 점검 진행 상황 실시간 표시
- 메트릭 실시간 업데이트
- 웹소켓 또는 폴링 (폴링으로 시작)

## Acceptance criteria

- [ ] 1분 간격 자동 새로고침 구현
- [ ] 수동 새로고침 버튼 구현
- [ ] 점검 진행 상황 실시간 표시 (진행 중 → 완료)
- [ ] 메트릭 실시간 업데이트 (서버 수, 상태 분포)
- [ ] 점검 중 새로고침 중단 가능
- [ ] 컴포넌트 언마운트 시 정리 (메모리 누수 방지)
- [ ] 사용자가 페이지를 떠났을 때 새로고침 중지
- [ ] 네트워크 오류 시 재시도 로직
- [ ] 시각적 피드백 (새로고침 중 표시)
- [ ] 단위 테스트: 새로고침 로직 검증
