## What to build

React 대시보드에서 보안 현황을 한눈에 파악할 수 있도록 시각화합니다. 메인 대시보드에는 전체 서버 수, 상태별 분포(정상/경고/위험), 취약점 심각도 분포(파이 차트) 등 핵심 메트릭을 표시합니다. 서버 목록은 상태별로 필터링 가능하며, 각 서버를 클릭하면 해당 서버의 모든 취약점을 조회할 수 있습니다. 취약점 목록에서는 심각도로 필터링하고, 각 취약점을 클릭하면 상세 정보(CVE, CVSS, 영향 범위, 조치 방법)를 볼 수 있습니다.

**핵심 요소:**
- 메인 대시보드 (메트릭: 서버 수, 상태 분포)
- 심각도별 분포 차트 (파이 차트)
- 서버 목록 페이지 (상태 필터링)
- 서버 상세 페이지 (취약점 목록)
- 취약점 목록 페이지 (심각도 필터링)
- 취약점 상세 페이지 (CVE, CVSS, 조치 방법)

## Acceptance criteria

- [ ] 메인 대시보드 구현 (전체 서버 수, 상태별 분포 표시)
- [ ] 취약점 심각도 분포 차트 (파이 차트)
- [ ] 서버 목록 페이지 (IP, 호스트명, OS, 점검 시간, 상태)
- [ ] 서버 상태별 필터링
- [ ] 서버 클릭 → 취약점 목록 표시
- [ ] 취약점 목록 페이지 (CVE, 제목, 심각도, 영향 서버 수)
- [ ] 취약점 심각도별 필터링
- [ ] 취약점 클릭 → 상세 정보 표시
- [ ] 상세 정보: CVE, CVSS 점수, 영향 범위, 한국어 조치 방법
- [ ] NVD 링크 제공
- [ ] 반응형 디자인 (모바일 지원)
- [ ] 통합 테스트: 전체 UI 흐름 검증
